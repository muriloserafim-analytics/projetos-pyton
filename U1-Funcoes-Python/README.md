# Student Grades Management System

Sistema simples de gerenciamento de notas desenvolvido em Python como atividade prática da disciplina de Linguagem de Programação.

## Funcionalidades
- Cadastro de notas
- Cálculo da média
- Verificação de aprovação ou reprovação
- Exibição de relatório final

## Tecnologias utilizadas
- Python
- Funções
- Estruturas condicionais
- Listas

## Objetivo
Aplicar conceitos básicos de programação utilizando Python.

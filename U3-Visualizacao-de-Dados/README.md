# Sales Data Analysis

Projeto de análise e visualização de dados desenvolvido em Python como atividade prática da disciplina de Linguagem de Programação.

## Funcionalidades
- Conexão com banco de dados SQLite
- Manipulação de dados com Pandas
- Análise de vendas por categoria
- Visualização de dados com Matplotlib
- Geração de gráficos com Seaborn

## Tecnologias utilizadas
- Python
- SQLite
- Pandas
- Matplotlib
- Seaborn

## Objetivo
Aplicar conceitos de:
- Banco de dados
- Manipulação de dados
- Análise de informações
- Visualização de dados
- Bibliotecas Python
